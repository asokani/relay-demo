#Relay demo

##Desired features
 
- Relay & GraphQL
- Redux
- hot reloading
- ~~isomorphic~~ universal
- ...

##Instructions

```
git clone https://github.com/petrhanak/relay-demo.git
npm install
npm run start
```
1. open `http://localhost:3000` in your browser
2. edit `src/router/index.js`
3. refresh browser
4. watch the magic
